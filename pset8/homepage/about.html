About this site

This four-page website was built for Harvard CS50x Problem Set 8 (Homepage). It satisfies the assignment's requirements:

  • Four linked pages (Home, Facts, Quiz, About).
  • The Bootstrap 5 front-end framework, loaded from a CDN.
  • A custom stylesheet (styles/style.css) with its own rules layered on top of Bootstrap.
  • Interactive JavaScript (scripts/main.js): a live footer clock and a client-side-graded quiz.

It is an independent educational project — the CS50 duck and CS50 name belong to Harvard University's CS50 team.

Tech stack
  • HTML5
  • CSS3 + Bootstrap 5.3
  • Vanilla JavaScript
